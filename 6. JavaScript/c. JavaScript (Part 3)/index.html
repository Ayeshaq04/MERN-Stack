String Methods

Strings are immutable : Means can't be changed, if once saved.

str.trim: Removes all useless whitespaces in a code [from start n end part only, spaces if in the middle won't be removed by this method] And this creates new str if run again, it will show the previous/original form.

str.toUpperCase: This helps in making all the text/content into CAPS

str.toLowerCase: This helps in making all the text/content into small-caps

String M with Arguments

str.indexOf(arg): Here, we can find out or search for any part/index in a given string. It'll show us the 1st occurence of that alphabet. If it doesn't find it, it'll show us (-1, means not found).

str.trim().toUpperCase(): This helps in chaining all the required methods together.

str.slice(): This helps in slicing the words from wherever we want it to slice. But we would have to add a starting n ending index for this process.

str.replace(): Searches a value in the string & returns a value replaced.

str.repeat(): This repeats with the number of copies mentioned.

Constant Array

Using this keyword 'const' would make the values unchangable. But if are using it with arr, we can still change/modify the values.
const arr = ['a', 'b'. 'c'];
arr.push('d');
arr.pop();
~ All of the above mentioned methods would work here in arr.
~ But adding the new array won't work, arr = ['a', 'b'. 'd'];
~ arr ke aage const likhne se uske values const nai hoti, but uska address nai change krwaya ja sakta.
