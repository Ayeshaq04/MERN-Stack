Grid Model

{Grid line: lines forming because, merging of rows n columns. Grid cell: box made from Grid lines. Grid track: space between 2 grid cell }

Grid Template

{Used for defining lines n tracking size. 2 sub-types: 1. grid-template-rows: 2. grid-template-columns: }

Grid Gaps

{Helps in adding some amount of gap in between elements/content. 1. row-gap: 10px; 2. column-gap: 10px; 3. grid-gap: 20px; [this value will apply to both row n column ]}

Grid Columns

{defines an item's starting n ending position.}

Grid Rows

{defines an item starting n ending position inside a row.}

Grid Properties

justify-item; {used for whole group} justify-self {used for individual elem} [used for horizontal layouts] align-item; {used for whole group} align-self {used for individual elem} [used for vertical layouts]

% Animation

If we want to change our property at every-other stage. Instead of using the {from} n {to} property. We can use this percent-property like 0%, 50%, 100%

Z-index

Used when we want 2 obj/elem to come on above the other. The only thing we need to have while using this property is that we must have a position property set beforehand. This property doesn't work when its set to Default or Static.

z-index: 1, n for the other obj if we set it to 2, the second obj will come over the 1st obj n vice-versa.
